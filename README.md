# Student Management System

## Overview

Student Management System is a Spring Boot REST API application that performs CRUD (Create, Read, Update, Delete) operations on student records.

The application uses Spring Data JPA and Hibernate for database interaction and MySQL as the backend database.

---

## API Endpoints

### Create Student

POST /students

### Get All Students

GET /students

### Get Student By ID

GET /students/{id}

### Update Student

PUT /students/{id}

### Delete Student

DELETE /students/{id}
